-------------------------------------------------------------------------------
CCFE plugin sample - Add a system resources monitoring menu.
-------------------------------------------------------------------------------

Description:
  This plugin is installed by default as demo and add some generic forms to
  call sar(1), iostat(1), vmstat(1), top(1) commands.
  They are made for the version of the commands available in the Ubuntu 9.10
  Linux distribution, so different O.S. and Linux distributions may not
  recognize all options.

Prerequisites:
  You must have the dynamic CCFE demo menu (a directory named "demo.menu")
  installed by default.
  ccfe program path must be in your PATH environment variable.

Installation:
  1. Go into ccfe-plugin-sysmon subdirectory in CCFE docs & samples directory
  2. Run the command 
                      ./install.sh

Usage:
  1. Run the command "ccfe demo" from a shell
  2. select the "CCFE demo menu" item
  3. select the "System resources usage monitors" item

  OR

  1. Run the command "ccfe sysmon" from a shell to use the plugin shortcut
